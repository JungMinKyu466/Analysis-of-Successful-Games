# 📊 프로젝트 제목

Steam 게임 데이터를 활용한 성공 요인 분석 프로젝트

---

## 📌 프로젝트 개요

프로젝트 목표:

- 시각화를 통해 성공한 게임의 특성들을 확인
- 데이터 시각화 기반 인사이트 도출

진행 기간:

- 2025.11 ~ 2025.12

진행 인원:

- 1명

---

## 📂 데이터 정보

데이터 출처:https://www.kaggle.com/datasets/fronkongames/steam-games-dataset

- 데이터명: Steam Games Dataset
- 출처:Kaggle

변수명 : 설명 

Name : 게임 이름
Estimated owners : Steam spy에서 추정한 게임 소유자의 추정치 
Peak CCU : 동시간대 최고 접속자 
Metacritic score : 전문 게임 매체의 평론가 점수. 0은 값이 없음 을 뜻함
Positive : ‘긍정’ 투표 수
Negative : ‘부정’ 투표 수
Average playtime forever : 2009년 이후의 평균 전체 플레이 시간
Required age : 게임 플레이 시 요구되는 나이. 0은 전체 연령을 뜻함
Price : 게임의 가격. 달러 단위
Supported languages : 게임이 지원하는 언어들
Genres : 게임의 장르

## 📈 주요 결과

### 결과 1

성공 게임 그룹(goat)로 갈 수록 전체 연령가 비중이 낮아지고 16세 이상 이용가 비중이 높아짐

> 단순 접근성을 위하여 연령 제한을 낮추기 보다, 16세 제한 이상의 겜임 컨셉에 맞는 연출을 하는 것이 효과적

---

### 결과 2

성공한 게임일 수록 저가형의 비중이 낮고, 표준 및 프리미엄 가격대의 비중이 높음

> 게임의 성공을 위해서는 저가형 보다는, 충분한 투자를 통해 퀄리티를 높이는 것이 효과적

---

### 결과 3

성공한 게임들은 영어 외에 독일어, 프랑스어, 스페인어, 이탈리아어를 공통으로 지원하는 경향을 보임

> 한정된 자원안에서 지원 언어를 선택할때, 위의 언어를 우선 고려하는 것이 효과적

---

### 결과 4

액션 장르가 가장 압도적이며, 인디, 어드밴처, 전략, RPG가 최신 트랜드임

> 액션 장르가 가장 대중적이고, 최신 유행하는 장르는 위와 같음
